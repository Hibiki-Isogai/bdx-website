## BDX への改善提案（調査結果ベース）

### P0: 即座に取り組むべき改善

1. **デザイン・視認性の向上**
   - 文字量を減らし、図・写真・動画を増やす
   - アニメーション・トランジションの追加（KPMG参考）
   - 青基調のデザインは BDX にも合う（ベイカレント参考）

2. **社員インタビューの充実**
   - 現在4名 → 最低8-10名に増やす（性別・ランク・部署の多様性）
   - 「働き方」インタビューを追加
   - 学生による社員インタビュー形式も検討（アビーム参考）

3. **キャリアパスの明示**
   - 入社後のステップを可視化
   - 1年目の働き方コンテンツ（EY参考）

### P1: 中期で取り組む改善

4. **選考フロー・面接対策ページ**
   - 選考プロセスの明確化
   - 準備のアドバイス（BCG・マッキンゼー参考）

5. **インサイト・ブログ**
   - 知見共有ページ（PwC のコラム参考）

6. **サイト構造の整理**
   - ベイカレントのような明確なナビゲーション
   - 情報のまとまりを意識

## スクレーピング対象（デザイン・実装分析）

以下5社を実際にスクレーピングし、CSS/アニメーション/レイアウト手法を分析:

1. **PwC** — https://www.pwc.com/jp/ja/careers/consulting/campus.html
2. **ベイカレント** — https://www.baycurrent.co.jp/careers/
3. **EY** — https://www.ey.com/ja_jp/careers
4. **KPMG** — https://recruit.kpmg-consulting.jp/new-graduate
5. **アビーム** — https://www.abeam.com/jp/ja/recruit/

分析観点:
- CSS アニメーション・トランジション手法
- セクション構成・レイアウトパターン
- ヒーローセクションの実装
- カラースキーム・タイポグラフィ
- モバイル対応手法
- インタラクション（スクロール連動、ホバーエフェクト等）
