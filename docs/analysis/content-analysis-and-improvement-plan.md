## 3. 改善計画

### Phase 1: 緊急修正（架空データの除去・実データ反映）

#### T-220: 統計数値の修正
- 根拠なき%数値を全て除去または「※」注記で暫定表記
- 81%（第一志望群）は出典確認後、正確な値に更新
- コンサルタント数を最新値に更新

#### T-221: 学生の声を実データベースに刷新
- 社員アンケートの実回答を匿名化して引用
- 以下のカテゴリで構成:
  1. BDXを選んだ理由（戦略コンサル環境、少数精鋭、給与、成長性）
  2. 選考で感じたこと（ケース面接のユニークさ、FBの質、社員の熱さ）
  3. インターンで学んだこと（実案件、スピード感、一緒に考えるスタイル）
  4. BDXの特徴（ESが短い、面接官が現役コンサル、対話重視）

#### T-222: インターン体験記の修正
- 5日間→2〜3日間に修正
- 架空の体験談を削除、実データベースの内容に差替

#### T-223: 選考フローページの精緻化
- 28卒の正確なフロー確認（1-2次同日 vs 1-3次同日）
- 社員アンケートの「BDXの選考の特徴」データを活用
- 「ケース面接が短時間×複数回」「面接官と一緒に考える」を明記

### Phase 2: メッセージング強化

#### T-224: トップページのメッセージ改善
- アンケートで刺さった要素を前面に:
  - 「社員の熱さ」「FBの質」「少数精鋭で埋もれない」
  - 「初任給640万円」（給与の透明性）
  - 「ESが短い」「選考が効率的」
- Numbers セクションを実データに更新

#### T-225: キャリアパスページの精緻化
- 適性検査データ（25卒内定承諾者の特性）を活用
  - 内定者の特徴: 説得力・指導力が高い、協議性が高い、面倒み良い
  - 一貫性が非常に高い(9.5)、上昇志向が高い(7.3)
- 「どんな人が活躍するか」のセクション追加

#### T-226: 情報不足問題への対応
- アンケートで指摘された「BDXの情報が少ない」への回答
  - FAQ強化: 仕事内容、キャリアパス、社風について踏み込んだ回答
  - 「コンサル業界のリアル」的なコンテンツ（守秘義務の範囲で）

### Phase 3: 新規コンテンツ追加

#### T-227: 認知経路を意識した訴求
- 外資就活ドットコム経由の学生が多いことを踏まえ:
  - 「戦略コンサル」「少数精鋭」キーワードの強化
  - 競合（デロイト、PwC、BCG等）との差別化ポイント明記

## 4. 実データに基づくコンテンツ素材

### 4-1. 学生の声（実データから匿名化引用）

**エントリー理由系**
- 「少数精鋭の環境に魅力を感じた。戦略コンサルとしてのプロジェクトが自分のやりたいことに近い」
- 「初任給640万円と記載されており、他社と比較して魅力を感じた」
- 「新進気鋭のコンサルティング企業であり、急成長している点に魅力を感じた」
- 「外資就活ドットコムでスカウトメールが届き、企業概要を読んで興味を持った」
- 「エントリーのハードルが低く、短文の自己PRなどで応募可能で気軽に応募できた」

**選考特徴系**
- 「ケース面接が短時間かつ回数が多い」
- 「面接官と一緒に考える形式で進むため、圧迫感が少ない」
- 「1~3次のケース面接が1日で終わるサバイバル感がある」
- 「選考回数が多く、様々な面接官と話す機会があるため、企業の雰囲気が掴みやすい」
- 「インターンのFBが非常に丁寧で、就活生の成長につながる」
- 「社員のレベルが高い（FBが良い、熱い）」
- 「就活生に対して忖度しない姿勢があり、能力だけでなく個人としての特性も見てくれる」

**入社決定系**
- 「会社を理解してもらおうとする姿勢が強い」
- 「代表取締役社長がインターンに参加し、企業のカルチャーを直接伝える」
- 「ESが短く、WEBテストやGDを実施せず、いきなり現役コンサルタントと面接ができる」

### 4-2. 就活で苦労した点（コンテンツのヒント）

- 「どの企業もキラキラしてて、実態が分かりにくい」
- 「コンサル業界は守秘義務があるため内情や働き方がわかりづらい」
- 「BDXについてネットで公開されている情報が本当に少なく、AB時代の記事を探して読み漁った」
- 「もう少し就活生にコミットした記事があったほうがいい」

→ **これらの声に応えるコンテンツを作ることが最大の訴求**

### 4-3. ファネルデータから見える特徴

- ES合格率が非常に高い（96%）→ 「まず出してみて」の訴求が有効
- 1次面接出席率に課題（66%）→ 面接予約を促す導線設計が重要
- 大学別: 東大の3次合格率が最も高い（86.5%）、慶應は数が多いが合格率は標準的

### 4-4. 内定承諾者の適性検査プロファイル

活躍人材の特徴（母集団平均との差）:
- **強み**: 協議性(+1.06)、面倒み(+1.26)、創造的(+1.08)、説得力(+0.81)、指導力(+0.84)
- **特徴**: 一貫性が非常に高い(9.5)、概念性が高い(8.5)
- **低い傾向**: 独自性(-0.98)、データ(-0.83)、人間(-1.01)

→ 「こんな人が活躍する」コンテンツに活用可能
