## 2. ページ別: 何を伝えているか

### 2-1. トップページ（recruitment-graduate/）

**ヒーロー**: 「戦略を描く。そして、実行する。」

| セクション | 伝えているメッセージ | 根拠データとの整合 |
|-----------|-------------------|-----------------|
| About B&DX（3カード） | 少数精鋭・戦略×実行・3年で10年分の成長 | ◎ アンケートの「少数精鋭」「戦略×実行」と一致 |
| 数字で見る | 2015年設立 / 50+名 / 81%第一志望群 / 100%継続率 | △ 81%の出典不明、50+は要確認 |
| 学生の声（3件） | ケース面接の対話感、FBの質、代表の参加 | ◎ アンケート実データに差替え済み |
| 成長のステージ | 3段階キャリアパス | ○ |
| 採用フロー | ES→1-2次→3次→JOB→役員→社長 | △ 28卒フロー要確認 |
| 募集要項 | 月給393,050円 | △ 年収レンジ非公開（アンケートでは給与が4位の入社理由） |

**このページの役割（現状）**: BDXの全体像を掴ませる入口。「何の会社か」「なぜここで働くべきか」を俯瞰的に伝える。

**データとのギャップ**:
- アンケートで4位の入社理由「給与・待遇（640万円）」がトップページに出てこない
- 「ESが短い・記載量が少ない」は selection に書いてあるが、トップの採用フローでは伝わらない
- 81%の出典が不明確

---

### 2-2. 学生の声（voices/）

**ヒーロー**: 「私たちの声が、あなたの一歩を後押しする。」

| セクション | 伝えているメッセージ | 根拠データとの整合 |
|-----------|-------------------|-----------------|
| ヒーロー統計（3つ） | ES合格率96% / 6年連続成長 / コンサル第一志望73% | ◎ 25卒実績データに基づく |
| 実績データ（4つ） | 96% / 1,100+ / 81% / 73% | △ 81%は出典確認中 |
| テーマ別の声（6件） | きっかけ/選考印象/FB/決め手/特徴/カルチャー | ◎ アンケート実データ |
| インターン体験記（2件） | 3日間の体験、実ビジネス課題、本気のFB | ○ 日数は修正済み |
| FAQ（6問） | ケース対策/文理/内定/面接回数/キャリア/ES | ○ |

**このページの役割（現状）**: 「先輩の証言」で感情的に背中を押す。数字で安心感、声で共感。

**データとのギャップ**:
- 6テーマの選定は良いが、アンケートで最も多い「戦略コンサルとしての環境」（7名）が独立テーマになっていない
- 「就活で苦労した点」（情報が少ない、内情がわかりにくい）に共感するコンテンツがない
- 競合エントリー先（デロイト9、PwC8、BCG5等）との比較視点がない

---

### 2-3. インターン（internship/）

**ヒーロー**: 「戦略を、体験する。」

| セクション | 伝えているメッセージ | 根拠データとの整合 |
|-----------|-------------------|-----------------|
| 概要（3カード） | 2-3日間 / 実課題 / メンター制 | ○ |
| タイムライン | Day1→Day2→Day3 の流れ | ○ |
| 参加者の声（3件） | 実課題、本気のFB、代表参加 | ◎ アンケートベース |
| 統計（3つ） | ES合格率96% / 1,100+エントリー / 2015年設立 | ◎ |
| FAQ（5問） | 選考/未経験/内容/優遇/交通費 | ◎ 新設 |

**このページの役割（現状）**: インターンの中身を可視化し、「行ってみたい」を作る。

**データとのギャップ**:
- インターン→本選考の具体的な優遇内容が曖昧（「一部免除等」のみ）
- 過去のインターンテーマ例がない（守秘義務の範囲で抽象化可能か？）
- 参加人数/倍率の情報がない

---

### 2-4. キャリアパス（career/）

**ヒーロー**: 「あなたのキャリアを、戦略的に。」

| セクション | 伝えているメッセージ | 根拠データとの整合 |
|-----------|-------------------|-----------------|
| 5段階タイムライン | アナリスト→パートナーまでの明確な道筋 | ○ |
| 1年目の働き方 | 09:00 MTG → 13:00 クライアント → 18:00 レビュー | ○ |
| 成長支援（3カード） | メンター / 研修 / 評価制度 | ○ |
| 活躍人材の特徴（3カード） | チーム力/説得力・一貫性/創造的問題解決 | ◎ 適性検査データ活用 |
| 社員インタビュー（3件） | 1年目から最前線、泥臭さ、育成文化 | ◎ |
| FAQ（5問） | スケジュール/裁量/研修/社風/戦略vs総合 | ◎ 新設 |

**このページの役割（現状）**: 「入社後の不透明さ」を解消。年次ごとの明確なゴールで安心させる。

**データとのギャップ**:
- タイムラインの年次に矛盾あり（マネージャーが「5年目〜」と「7年目〜」で不整合）
- プロジェクト事例がゼロ（「実際にどんな仕事をするのか」が見えない）
- 給与テーブル/昇給モデルがない（アンケートで給与は4位の入社理由）

---

### 2-5. 選考フロー・面接対策（selection/）

**ヒーロー**: 「選考は、対話の場。」

| セクション | 伝えているメッセージ | 根拠データとの整合 |
|-----------|-------------------|-----------------|
| 6ステップフロー | Entry→ES→1次→2次→3次→内定 | △ 28卒フロー要確認（JOB/役員/社長がない） |
| 面接で見るポイント（3つ） | 論理的思考/コミュニケーション/成長意欲 | ◎ アンケートの選考特徴と一致 |
| 面接対策（3カテゴリ） | ES/ケース/最終の具体Tips | ○ |
| FAQ（8問） | 実践的な不安解消 | ◎ 「他社との違い」「情報が少ない」に直球回答 |
| 先輩メッセージ（2件） | 面接官側の視点開示 | ◎ |

**このページの役割（現状）**: 選考の心理的ハードルを下げる。「評価される場」→「対話の場」にフレーミング転換。

**データとのギャップ**:
- トップページの採用フロー（7ステップ: ES→1-2次→3次→JOB→役員→社長）と、selectionの6ステップが食い違っている
- 「1〜2次を同日実施」がselectionでは明記されているがトップページでは不明確
- 具体的な所要期間（ES提出〜内定まで何週間?）がない

## 3. ページ間の導線（現状の学生体験フロー）

```
学生の典型的な回遊パターン（推測）:

外資就活/ワンキャリア/スカウト
    ↓
[トップページ] 「BDXって何？」→ 全体像を掴む
    ↓
[学生の声] 「実際どうなの？」→ 先輩の証言で安心
    ↓                    ↓
[インターン] 「体験したい」  [選考フロー] 「面接はどんな感じ？」
    ↓                    ↓
[キャリアパス] 「入社後はどうなる？」
    ↓
エントリー（Sonar）
```

**問題**: 各ページにCTAはあるが、ページ間の導線が弱い。「次にどこを見ればいいか」が明示されていない。学生は自力でナビゲーションから探す必要がある。

## 4. データとの突合サマリ: 何が足りないか

### 伝わっていること ✅

| メッセージ | 根拠 |
|-----------|------|
| 戦略×実行の一気通貫 | アンケート47%が言及 → サイト全体で繰り返し訴求 |
| 社員の本気度・FBの質 | アンケート多数言及 → voices, selection で証言 |
| 少数精鋭で埋もれない | アンケート「エントリー理由」1位 → top, career で訴求 |
| 選考のハードルが低い | ES96%, WEBテストなし → selection, voices で明記 |

### 伝わっていないこと ❌

| メッセージ | データ根拠 | 現状 |
|-----------|-----------|------|
| **給与の魅力（640万円）** | アンケート入社理由4位 | 募集要項に月給のみ。年収未掲載 |
| **ESの手軽さ** | アンケート「ESが短い」3名 | selection FAQに少し触れるだけ |
| **就活生の不安への共感** | 「情報が少ない」「内情がわかりにくい」 | FAQ1問で言及のみ。共感コンテンツなし |
| **競合との明確な差別化** | デロイト9名、PwC8名がエントリー先 | 「大手コンサルとは違う」程度。具体的な比較なし |
| **プロジェクト実例** | 企業研究で重視1位「実際の仕事内容」(12名) | ゼロ。守秘義務の壁 |
| **社風の具体像** | 企業研究で重視2位「社風」(10名) | FAQ・声で断片的に伝わるが、専用コンテンツなし |
| **認知経路の最適化** | 外資就活53%が入口 | サイト内に「外資就活から来た方へ」的な導線なし |

## 6. 現状の強みと弱み

### 強み
1. **実データベースのコンテンツ**: Phase H3で架空データを排除し、アンケート実データに差替え済み
2. **選考ハードル低下のメッセージ**: ES96%通過、WEBテストなし、ケース未経験OKを繰り返し伝えている
3. **社員の熱量の可視化**: FB、忖度しない姿勢、代表のインターン参加が複数ページで言及
4. **FAQ充実**: 全5ページに合計29問のFAQ（selection8問, voices6問, internship5問, career5問, 計24問 + その他）

### 弱み
1. **給与情報の不透明さ**: 学生が4番目に重視する要素なのに、月給しか出していない
2. **プロジェクト事例ゼロ**: 学生が最も知りたい「実際の仕事内容」に答えていない
3. **ページ間導線の弱さ**: 各ページが独立しており、ストーリーとしてつながっていない
4. **競合との差別化が曖昧**: 「大手とは違う」以上の具体的な比較がない
5. **情報不足問題への対応が表面的**: FAQ1問で終わっており、根本的な解決（コンテンツの深さ）に至っていない
6. **トップページと選考ページのフロー不整合**: 7ステップ vs 6ステップの食い違い
