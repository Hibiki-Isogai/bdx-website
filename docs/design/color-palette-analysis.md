# カラーパレット分析レポート

> T-237: 採用ページのカラーパレットを基準ページに統一する
> 作成日: 2026-03-16

---

## 背景

Owner 指摘: 採用ページ（recruitment-graduate/）の色使いが BDX サイト他ページと比べてトーンが合っていない。
- 青が鮮やかすぎる
- 青以外のカラー（黄色系など）が安易に使われている
- 超アテンション（赤など）以外は他ページと統一すべき

## 3. ギャップ一覧

### A. 青系カラーの不一致（最重要）

| 項目 | 基準ページ | 採用ページ | 問題 |
|---|---|---|---|
| メインブルー | `#1A407B` (彩度64%) | `#003894` (彩度100%) | **彩度が 64% → 100%で鮮やかすぎる** |
| 補助ブルー | `#004B9F` | `#1a5cc8` | 明るく鮮やか |
| ダーク背景 | なし | `#002266`, `#0A1628` | 基準にはない濃紺グラデーション |

### B. ゴールド系カラー（基準に存在しない）

`#C8A864` がCTAボタン背景・タイムラインマーカー・ラベル装飾・引用ボーダー等に多用されている。
基準サイトにゴールド/黄色系は一切存在しない。

### C. テキスト・ボーダーの微差

独自値を定義しているため、並べると統一感が損なわれている。

### D. グラデーション

基準ページの共通CSSには `linear-gradient` が一切ないが、採用ページでは多用。
グラデーション自体は差別化として許容できるが、色コードが基準と合っていないのが問題。

---

## 4. 根本原因

採用ページは基準サイトの共通CSS (`styles.css`) のカラーパレットを参照せず、
独自の CSS 変数セット (`--bdx-navy`, `--bdx-gold`) を定義している。

CSS変数で管理されているため、各ページの `:root` セクション内の値を書き換えれば大部分は解決する。
