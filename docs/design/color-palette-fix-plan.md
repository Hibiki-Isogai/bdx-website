# カラーパレット統一 修正計画

> T-237: 採用ページのカラーパレットを基準ページに統一する
> 作成日: 2026-03-16
> 前提: [color-palette-analysis.md](color-palette-analysis.md) の分析結果に基づく

---

## 修正方針

**基準ページ（共通CSS）のカラーパレットに合わせる。採用ページ独自のカラーは原則廃止。**

例外:
- グラデーション表現は採用ページの差別化として残す（ただし色は基準に合わせる）
- 超アテンション用途には既存の赤系 `#DD083C` を使用可

---

## 修正タスク一覧

### P0: 青系カラーの統一（全5ページ）

各ページの `<style>` 内 CSS変数定義を書き換え:

| CSS変数 | 現在 | 変更後 | 備考 |
|---|---|---|---|
| `--bdx-navy` | `#003894` | `#1A407B` | 基準のメインブランドブルー |
| `--bdx-navy-light` | `#1a5cc8` | `#004B9F` | 基準の補助ブルー |
| `--bdx-navy-dark` | `#002266` | `#0f2d50` | #1A407B を暗くした値 |

対象ファイル（5ファイル、各ファイル3行の変更）:
- `site/recruitment-graduate/index.html`
- `site/recruitment-graduate/voices/index.html`
- `site/recruitment-graduate/internship/index.html`
- `site/recruitment-graduate/career/index.html`
- `site/recruitment-graduate/selection/index.html`

追加対応:
- SVGアイコン内の `stroke="#003894"` / `fill="#003894"` → `#1A407B` に置換
- インラインスタイルの `#003894` ハードコード → `#1A407B` に置換

### P1: ゴールド系カラーの除去（全5ページ）

| CSS変数 | 現在 | 変更後 | 用途別の置換先 |
|---|---|---|---|
| `--bdx-gold` | `#C8A864` | 廃止 | 下記参照 |
| `--bdx-gold-light` | `#D4BC82` | 廃止 | 下記参照 |

用途別の置換先:

| 現在のゴールド使用箇所 | 置換先 | 理由 |
|---|---|---|
| CTAボタン背景 | `#1A407B`（基準のボタン色） | 基準ページのボタンに統一 |
| タイムラインのステップマーカー | `#1A407B` | ブランドブルーで統一 |
| セクションラベル `color: var(--bdx-gold)` | `#1A407B` or `#ffffff` | 背景色に応じて |
| 引用ボーダー `border-left` | `#1A407B` | ブランドブルーで統一 |
| タグ/チップ装飾 | `#EDF0F5` 背景 + `#1A407B` テキスト | 基準の薄い青グレー系 |
| box-shadow `rgba(200,168,100,*)` | `rgba(26,64,123,*)` | #1A407B のRGB値ベース |
| 直接指定 `#8a6a1a` | `#1A407B` | ブランドブルーで統一 |

### P2: テキスト・ボーダー色の統一（全5ページ）

| CSS変数 | 現在 | 変更後 |
|---|---|---|
| `--bdx-text-primary` | `#1A1A2E` | `#1A2229` |
| `--bdx-text-secondary` | `#4A5568` | `#4C5262` |
| `--bdx-border` | `#E2E8F0` | `#DEE1E3` |
| `--bdx-bg-light` | `#F5F7FA` | `#F5F6F7` |

### P3: グラデーション色の調整

グラデーション構造は維持し、色のみ基準パレットに合わせる:

| 現在 | 変更後 |
|---|---|
| `rgba(0,34,102,*)` | `rgba(26,64,123,*)` (#1A407B ベース) |
| `rgba(0,56,148,*)` | `rgba(26,64,123,*)` (#1A407B ベース) |

---

## 実装手順

1. **P0 + P2**: CSS変数の値を書き換え（5ファイル × 7変数 = 35箇所）
2. **P1**: ゴールド変数を廃止し、使用箇所を個別に置換
3. **P0 追加**: SVG / インラインスタイルのハードコード色を置換
4. **P3**: グラデーションの rgba 値を置換
5. **検証**: 全5ページの表示確認（ステージング）

## 見積もり

CSS変数による管理のため、変数定義の書き換えで大部分が一括反映される。
ゴールド廃止部分のみ、用途別に個別対応が必要。
